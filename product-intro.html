用户研究

痛点分析

创作者缺少适配移动端、低门槛且专为吧唧场景设计的工具

技能门槛高

  • 需要 Photoshop 等专业软件
  • 抠图、调色、排版需设计经验
  • 大量创作者望而却步

流程耗时

  • 手动抠图+找素材+排版 = 2-3小时
  • 调尺寸反复试错
  • 多款式需重复劳动

素材难寻

  • 边框素材质量参差不齐
  • 可爱风装饰散落各处
  • 版权问题模糊

工具不适配

  • 通用工具不懂吧唧规格
  • 无圆形预览+印刷标准
  • 手机端体验差

技术亮点

纯前端 · 零后端

  • React + TypeScript + Vite
  • 所有计算在用户设备本地完成
  • 图片永不上传服务器
  • 边际运营成本趋近于零

  • WebAssembly + ONNX Runtime
